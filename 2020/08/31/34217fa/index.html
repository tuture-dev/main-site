小结

在本文中，我们介绍了 mock 模块，并将其用于伪造 API 调用。 由于没有发起实际的 post 请求，我们的测试可以更可靠，更快。 除此之外，我们还在整个 React 组件中模拟了事件。 我们检查了它是否产生了预期的结果，例如组件的请求或状态变化。 为此，我们了解了 spy 的概念。
